# Quick Start Guide - Get Your Landing Page Live in 30 Minutes

## What You Have

A professional real estate lead capture landing page built with React that's ready to deploy. It includes:

- Beautiful hero section with your value proposition
- Lead capture form (name, phone, email, property type, condition, location)
- Property types showcase (single family, multifamily, self-storage, industrial-flex)
- "Why Choose Us" section
- Mobile-responsive design
- Professional dark theme with blue accents

## The 30-Minute Setup

### Step 1: Create a GitHub Account (5 minutes)

1. Go to https://github.com/signup
2. Enter your email address
3. Create a password
4. Choose a username (e.g., `your-name-real-estate`)
5. Verify your email
6. Click "Create account"

### Step 2: Upload Your Files to GitHub (10 minutes)

1. Log in to GitHub
2. Click the "+" icon (top-right) → "New repository"
3. Name it: `real-estate-leads`
4. Select "Public"
5. Click "Create repository"
6. GitHub will show you instructions to upload files
7. Follow the instructions to push your project files

**Quick command-line version:**
```bash
git init
git add .
git commit -m "Initial landing page"
git branch -M main
git remote add origin https://github.com/YOUR-USERNAME/real-estate-leads.git
git push -u origin main
```

### Step 3: Deploy to Netlify (10 minutes)

1. Go to https://netlify.com
2. Click "Sign up" → Choose "GitHub"
3. Authorize Netlify to access your GitHub
4. Click "New site from Git"
5. Select your `real-estate-leads` repository
6. Click "Deploy site"
7. **Your site is now live!** Netlify will give you a temporary URL

### Step 4: Set Up Form Submissions (5 minutes)

1. Go to https://web3forms.com
2. Click "Get Started"
3. Copy your access key
4. Open your landing page code and find: `YOUR_WEB3FORMS_KEY_HERE`
5. Replace it with your actual access key
6. Commit and push the change to GitHub
7. Netlify will automatically redeploy your site

## Connect Your Domain

Once your site is live, connect your existing domain:

1. Log in to Netlify
2. Go to your site settings
3. Click "Domain settings"
4. Click "Add custom domain"
5. Enter: `webuyrealestateanycondition.com`
6. Netlify will show you DNS records to add
7. Go to your domain registrar (GoDaddy, Namecheap, etc.)
8. Add the DNS records Netlify provided
9. Wait 24-48 hours for DNS to update

## Test Your Form

1. Go to your live website
2. Fill out the form completely
3. Click "Get Your Free Offer"
4. You should see a success message
5. Check your email for the form submission

## Troubleshooting

**Form not working?**
- Make sure you added your Web3Forms access key
- Check your spam folder for test submissions
- Try clearing your browser cache

**Domain not connecting?**
- Wait 24-48 hours for DNS to propagate
- Check DNS records at https://mxtoolbox.com/dnscheck
- Contact your domain registrar if issues persist
