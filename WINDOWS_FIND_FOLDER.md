## EASIEST METHOD - Use the Address Bar

This is the fastest way:

### Step 1: Open File Explorer
- Click the **Windows Start button** (bottom-left corner of your screen)
- Look for **File Explorer** icon (looks like a folder)
- Click it to open

### Step 2: Click the Address Bar
- At the very top of File Explorer, you'll see a bar with text (might say "C:\" or "This PC")
- Click in that bar to select it
- The text should turn blue (highlighted)

### Step 3: Paste the Folder Path
- Copy this text: `C:\Users\ubuntu\real_estate_leads`
- Right-click in the address bar
- Click **Paste**
- The path should now appear in the address bar

### Step 4: Press Enter
- Press the **Enter** key on your keyboard
- Your folder will open!

### Step 5: Verify You Found It
- You should see these folders/files inside:
  - **client** (folder)
  - **server** (folder)
  - **shared** (folder)
  - **package.json** (file)
  - **QUICK_START.md** (file)
  - **DEPLOYMENT_GUIDE.md** (file)

## ALTERNATIVE METHOD - If Address Bar Doesn't Work

If the address bar method doesn't work, try this:

### Step 1: Open File Explorer
- Click **Windows Start** button
- Type: `File Explorer`
- Press **Enter**

### Step 2: Navigate to Users Folder
- Look on the left side for **"This PC"** or **"My Computer"**
- Click it
- You should see drives like **C:**, **D:**, etc.
- Double-click the **C:** drive

### Step 3: Find Users Folder
- Look for a folder called **"Users"**
- Double-click to open it

### Step 4: Find ubuntu Folder
- Look for a folder called **"ubuntu"**
- Double-click to open it

### Step 5: Find real_estate_leads Folder
- Look for a folder called **"real_estate_leads"**
- Double-click to open it
- This is your folder!
